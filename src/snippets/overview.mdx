<p class="t-lead">LearnUXD brings you this course on basic User Experience (UX) and Interface Design. Learn from some of the leading experts in the field of User Experience about usability, user research, and the user-centered approach to designing digital interfaces and products.</p>

By the end of this course students will be able to:

- Explain concepts of effective user interaction design and apply them to plan and critique interactive interfaces for a variety of media.
- Develop their abilities with rapid sketching, particularly for the benefit of developing and testing interface concepts.
- Create prototype versions of interactive interfaces using concept sketches, wireframes, and static technologies that can be used for user testing and for demonstrations to clients and developers.
- Demonstrate an increased value for human-centered design and stewardship, especially from a Biblical perspective.
